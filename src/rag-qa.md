## 1. ระบบตอบกลับช้า / ระบบไม่ตอบกลับ

**Q: ส่งใบขนไปแล้วระบบไม่ตอบกลับ ต้องทำอย่างไร?**
A: ให้รอก่อนครับ ระบบกรมศุลกากรตอบช้าปกติรออยู่ที่ 10–30 นาที ถ้าเกิน 30 นาทีแจ้งเลขอ้างอิงให้ทาง CDS ตรวจสอบค่ะ

**Q: ระบบตอบกลับช้ากี่นาทีถือว่าปกติ?**
A: ปกติระบบกรมศุลฯตอบช้าอยู่ที่ 30 นาที ถ้าไม่เกิน 30 นาทีให้รอก่อน ถ้าเกิน 30 นาทีให้แจ้งเลขอ้างอิงเพื่อตรวจสอบค่ะ

**Q: ระบบ อย. ตอบกลับช้า ต้องทำอย่างไร?**
A: ระบบ อย. ตอบกลับช้า ต้องรอคำตอบก่อนค่ะ ทาง CDS จะประสานงาน NSW ให้

## 2. การส่งซ้ำ (Resend)

**Q: ส่งใบขนไปแล้วไม่ได้รับคำตอบ จะส่งซ้ำได้อย่างไร?**
A: ไปที่เมนู แลกเปลี่ยนข้อมูล → ส่งไปแล้ว → เลือกเบอร์อ้างอิง → ส่งซ้ำ ค่ะ

**Q: ใบขนส่งเกิน 10 ครั้งแล้ว ทำอย่างไร?**
A: ต้องทำสำเนาเป็นเลขอ้างอิงใหม่ แก้ไขแล้วส่งใหม่ค่ะ เพราะส่งเกิน 10 ครั้งจะไม่สามารถส่งซ้ำได้อีก

**Q: เมื่อไหรควรส่งซ้ำ และเมื่อไหรไม่ควรส่งซ้ำ?**
A: ควรส่งซ้ำเมื่อไม่มีเอกสารเข้าระบบและสถานะค้างนานเกิน 30 นาที ไม่ควรส่งซ้ำเมื่อได้รับคำตอบหรือได้เลขใบขนแล้ว เพราะการส่งซ้ำจะทำให้ขึ้น Reject และมีค่าใช้จ่ายทุกครั้งที่ส่งค่ะ

## 3. Car Manifest / Sea Manifest

**Q: ยิงคาร์ไปนานแล้วยังไม่ได้คำตอบ ระบบมีปัญหาไหม?**
A: Car manifest ตอบช้ารอสักครู่ค่ะ ถ้าเกิน 30 นาทีแจ้งเลขอ้างอิง CDS จะประสานงาน NSW และกรมศุลกากรให้

**Q: ไม่พบ Car Manifest / ขึ้น Carmanifest not found ทำอย่างไร?**
A: ต้องตรวจสอบ Manifest กับผู้ที่ส่ง Manifest หรือสอบถาม Agent ค่ะ ทาง CDS ตรวจสอบข้อมูล Manifest ไม่ได้

**Q: ยิงใบขนขาเข้าทางแอร์ตอบกลับว่า manifest not found ทำอย่างไร?**
A: ไม่พบข้อมูล manifest ให้ติดต่อ agent ค่ะ

**Q: ใบคาร์ที่ยื่นไว้แล้วไม่ได้ใช้ ต้องการยกเลิก ทำได้ในระบบไหม?**
A: ให้โทรติดต่อเข้าออฟฟิศสอบถามเพิ่มเติม 02-2544027-9 ค่ะ

**Q: ระบบ Car Manifest มีปัญหาวงกว้าง หลายคนส่งแล้วไม่ตอบกลับ?**
A: ทาง CDS ได้ประสานงานไปยัง NSW และเจ้าหน้าที่ Support Desk กรมศุลกากรแล้วค่ะ รอก่อนนะคะ จะแจ้งให้ทราบเมื่อมีความคืบหน้า

## 4. CONSIGNEE NAME NOT EQUAL MANIFEST

**Q: ส่งตั๋วแล้วระบบตีกลับว่า CONSIGNEE NAME NOT EQUAL MANIFEST หมายความว่าอะไร?**
A: ชื่อผู้นำเข้าในใบขนไม่ตรงกับจดทะเบียนกรมศุลฯ หรือชื่อภาษาอังกฤษที่ส่งใน Manifest ไม่ตรงกับฐานข้อมูลกรมฯ ให้ตรวจสอบชื่อกับ DO หรือสอบถามสายเรือ/สายการบินที่ลง Manifest ค่ะ

**Q: ชื่อผู้นำเข้าจดทะเบียนเป็น ABC (THAILAND) LTD แต่ดีโอของการบินไทยเป็น ABC THAILAND LTD ไม่มีวงเล็บ แบบนี้ส่งใบขนมีปัญหาไหม?**
A: แนะนำสอบถาม จนท กรมฯ น่าจะได้คำตอบที่ชัดเจนกว่าค่ะ

## 5. IMPORT TAX INCENTIVES ID — สิทธิประโยชน์

**Q: ขึ้น IMPORT TAX INCENTIVES ID (DETAIL) MUST NOT BE ENTERED หมายความว่าอะไร?**
A: ไม่ต้องระบุ Tax Incentives ID สำหรับใบขนนี้ค่ะ แต่ถ้าติดปัญหาให้แจ้งเลขอ้างอิงให้ จนท CDS ตรวจสอบ โทร 022544027-9

**Q: ขึ้น IMPORT TAX INCENTIVES ID MUST BE ENTERED หมายความว่าอะไร?**
A: ต้องใส่เลขสิทธิประโยชน์ในรายการสินค้าค่ะ ไปเพิ่มที่ช่องสิทธิประโยชน์ในรายการสินค้า

## 6. ค่าธรรมเนียม

**Q: Error GDEX-00305 ไม่ต้องระบุค่าธรรมเนียมการดำเนินพิธีการศุลกากร ติ๊กค่าธรรมเนียมแล้วส่งไม่ผ่าน ทำอย่างไร?**
A: ไม่ต้องติ๊กค่าธรรมเนียม 200 ค่ะ เพราะราคาสินค้าไม่ถึง 50,000 บาท ถือว่าเข้าเงื่อนไขไม่ต้องชำระค่าธรรมเนียมตามประกาศ

**Q: ต้องจ่ายค่าธรรมเนียม ติ๊ก A ชำระที่กรมศุล ได้ใบขนแล้ว ใบเสร็จจะได้ตอนไหน?**
A: ใบเสร็จต้องปริ้นใน e-tracking กรมศุลฯค่ะ ไม่ได้ปริ้นจากโปรแกรม CDS โดยตรง

**Q: ค้างค่าธรรมเนียมล่วงเวลาอยู่ ทำอย่างไร?**
A: ต้องไปจ่ายค่าธรรมเนียมเดือนเก่าที่กรมศุลฯก่อนค่ะ ติดต่อการเงินกรมศุลฯเพื่อตรวจสอบ

**Q: ขึ้น error ค้างค่าธรรมเนียมในกล่องรับ ทำรายการไม่ได้?**
A: ต้องติดต่อการเงินกรมศุลฯนะคะ เพราะค้างชำระค่าล่วงเวลาอยู่

## 7. อัตราแลกเปลี่ยน

**Q: ดาวน์โหลดอัตราแลกเปลี่ยนไม่ได้ หรือโหลดแล้วข้อมูลไม่อัปเดต ทำอย่างไร?**
A: ให้กดเลือกรายการที่มีปัญหา แล้วกดปุ่ม Del ที่คีบอร์ด ปิดแล้วดาวน์โหลดใหม่ โดยกดปุ่ม ดาวน์โหลด ไม่ใช่ปุ่ม MaskDownload ค่ะ

**Q: โหลดอัตราแลกเปลี่ยนแล้วแต่ใบกำกับไม่พบอัตราแลกเปลี่ยน ทำอย่างไร?**
A: ตรวจสอบว่าวันที่ที่ใส่ในใบกำกับเป็น ค.ศ. (2026) หรือ พ.ศ. (2569) ต้องใส่ให้ตรงกับที่ตั้งในเครื่อง ถ้าเครื่องเป็น 2026 ก็ต้องใส่ 2026 หากยังไม่ได้ให้ลองปิดโปรแกรมแล้วเปิดใหม่ หรือลบหน้าประวัติดาวน์โหลดแล้วโหลดใหม่ทั้ง 3 อัน (ขาเข้าและขาออก) ค่ะ

**Q: REJECT เรื่องอัตราแลกเปลี่ยนไม่ถูกต้อง ทำอย่างไร?**
A: คำนวณอัตราแลกเปลี่ยนใหม่แล้วส่งใหม่ค่ะ

**Q: เรือเข้าข้ามเดือน อัตราแลกเปลี่ยนเปลี่ยน มีหลายรายการ จะอัปเดตอัตราแลกเปลี่ยนทั้งหมดได้อย่างไร?**
A: ให้เปลี่ยนวันที่นำเข้าที่ Header แล้วกดคำนวณอัตราแลกเปลี่ยนใหม่ค่ะ

## 8. เข้าโปรแกรมไม่ได้ / Login ไม่ได้

**Q: เข้าโปรแกรม CDS Express ไม่ได้ เกิดจากอะไร?**
A: มักเกิดจากปัญหาถูกบล็อคจากโปรแกรมในเครื่อง เช่น โปรแกรมป้องกันไวรัส หรือ Smart App ของ Windows ให้ปลดบล็อคก่อน และตรวจสอบอินเทอร์เน็ตว่าใช้งานได้ปกติค่ะ

**Q: เครื่องลูกเปิดเข้าโปรแกรมไม่ได้หลังเปลี่ยน ISP ทำอย่างไร?**
A: ต้องใช้อินเทอร์เน็ตเดียวกันทั้งหมด (เครื่องแม่และเครื่องลูกต้องอยู่ network เดียวกัน) ค่ะ ถ้าทำแล้วยังไม่ได้ติดต่อ 02-2544027-9

**Q: เข้าโปรแกรมบนเครื่องลูกไม่ได้ ขึ้น error ทำอย่างไร?**
A: ตรวจสอบ Firewall เครื่องแม่ว่าปิดอยู่หรือไม่ (ปกติต้องปิด) และตรวจสอบ Network ด้วยค่ะ หากยังไม่ได้ติดต่อ 02-2544027-9

**Q: Password CA ผิด กด Send แล้วขึ้นสีแดง ทำอย่างไร?**
A: ตรวจสอบ Password CA ให้ถูกต้องแล้วลองใหม่ค่ะ

## 9. สถานะใบขน (02 / 03 / 04)

**Q: คืนตู้ที่ท่าเรือแล้วแต่ในระบบยังไม่เป็น 03 ทำอย่างไร?**
A: ต้องสอบถามที่ท่าเรือค่ะ ทาง CDS เห็นแค่สถานะที่มีอยู่ในระบบ สถานะ 03 ขึ้นอยู่กับการอัปเดตจากท่าเรือ

**Q: ทางเจ้าหน้าที่ด่านแจ้งว่าเป็น 04 แล้วแต่ในโปรแกรมยังเป็น 03 อยู่ ทำอย่างไร?**
A: ให้ลองเปิดตัวรับส่ง (Connect) แล้ว Refresh ใหม่ค่ะ สถานะจะเปลี่ยนเป็น 04 หลังรับข้อมูลจากระบบ

**Q: ใบขนขึ้นเลขที่แล้วแต่สถานะยังขึ้นรอคำตอบจากกรมศุล เพราะอะไร?**
A: เกิดจากมีการกดส่งซ้ำเข้ามาหลังจากได้รับเลขใบขนแล้ว เป็นแค่สถานะในโปรแกรม ใบขนสามารถใช้งานได้ตามปกติ ไม่ต้องกังวลค่ะ

**Q: วันที่ใน STATUS สลับกัน ปกติไหม?**
A: ปกติค่ะ เกิดจากการรับคำตอบหลังจากได้สถานะ 03 และ 04 แล้ว และวันที่ได้รับชำระค่าธรรมเนียมกรมศุลฯไม่มีวันที่มาให้ จึงใช้วันที่รับคำตอบแทน

**Q: ใบขนชำระเงินแล้วบางบริษัทได้ 02 บางได้ 03 ต่างกันอย่างไร?**
A: ให้ติดต่อออฟฟิศ 02-2544027-9 เพื่อสอบถามรายละเอียดเพิ่มเติมค่ะ

## 10. Error ต่างๆ ในใบขน

**Q: REJECT ลำดับอัตราอากรหมดอายุ ทำอย่างไร?**
A: ลำดับอัตราอากรที่ใช้หมดอายุแล้ว ต้องเลือกลำดับอัตราอากรที่ถูกต้องและยังไม่หมดอายุค่ะ

**Q: REJECT ลำดับอัตราอากรที่ใช้ยังไม่เริ่มใช้งาน ทำอย่างไร?**
A: ลำดับอัตราอากรที่เลือกยังไม่ถึงวันเริ่มใช้งาน ต้องเลือกลำดับที่ถูกต้องค่ะ

**Q: ขึ้น INVALID STATISTIC CODE รหัสสถิติ 000 ของพิกัดนี้ยังใช้ได้อยู่ไหม?**
A: กรมศุลฯตอบว่าใช้ไม่ได้แล้วค่ะ ต้องใช้รหัสสถิติที่ถูกต้อง สอบถาม จนท CDS หรือตรวจสอบจากเว็บกรมศุลฯ

**Q: ขึ้น error อักขระพิเศษในโปรแกรม ทำอย่างไร?**
A: มีการ Copy ข้อความที่มีอักขระพิเศษมาวางในโปรแกรม ให้ลบข้อมูลที่ copy มาวาง แล้วคีย์ใหม่เองค่ะ อย่า copy-paste จากเอกสารอื่นโดยตรง

**Q: ขึ้น error ไม่พบเลขที่ใบอนุญาต / ลำดับอากรที่เลือกไม่ถูกต้อง ทำอย่างไร?**
A: ให้กดปุ่มแว่นขยายที่ช่องรหัสสิทธิพิเศษ ดูว่า "ลำดับอากรที่เลือกไม่ถูกต้อง" แล้วกดคำนวณใหม่ค่ะ ถ้าใบอนุญาตไม่มีในระบบต้องตรวจสอบกับกรมฯ

**Q: ใบขนผ่านแดนขึ้น Reject แต่ได้เลขใบขนและใบเคลื่อนย้ายแล้ว ต้องทำอะไรไหม?**
A: ไม่ต้องทำอะไรค่ะ ขึ้น Reject เพราะมีการส่งซ้ำเข้ามาหลังจากได้เลขแล้ว ในระบบกรมศุลฯผ่านตั้งแต่รอบแรก สามารถใช้งานได้เลย

**Q: ข้อมูลแพ็คเกจไม่ตรงกับ Manifest ทำอย่างไร?**
A: ต้องตรวจสอบข้อมูลแพ็คเกจในใบขนว่าตรงกับข้อมูลใน Manifest หรือไม่ค่ะ

**Q: ITEM NO 1 DISCHARGE PORT MISMATCH หมายความว่าอะไร?**
A: สถานที่นำเข้า (discharge port) ไม่ตรงกับใบอนุญาต ต้องแก้ไขให้ตรงกันค่ะ

**Q: Total payment amount not equal total tax คืออะไร?**
A: ยอดชำระไม่เท่ากับภาษีทั้งหมด ให้เข้าไปดูในกล่องรับเพราะมักมี error อื่นด้วย เช่น วันเรือเข้าไม่ถูกต้อง ให้แก้ไขข้อมูลนั้นก่อนค่ะ

**Q: Error vsed-0061 คืออะไร ทำอย่างไร?**
A: ให้เข้า แลกเปลี่ยนข้อมูล → กล่องรับ เพื่อดู error รายละเอียดเพิ่มเติมค่ะ

**Q: REJ หรือ Reject เรื่องทะเบียนผู้นำเข้าส่งออกหมดอายุ ทำอย่างไร?**
A: ทะเบียนผู้นำเข้าส่งออกหมดอายุแล้ว ต้องต่ออายุทะเบียนกับกรมศุลกากรก่อนค่ะ

**Q: พิกัดสินค้าในเว็บกรมไม่ติดใบอนุญาต แต่ในใบขนแจ้งว่าต้องระบุใบอนุญาต แบบนี้ต้องระบุไหม?**
A: อิงตามเว็บกรมฯ เป็นหลักค่ะ ถ้าเว็บกรมไม่ได้ระบุว่าต้องใช้ใบอนุญาต ก็ไม่ต้องระบุ

**Q: ประเทศต้นทางไม่ตรงกับใบอนุญาต ทำอย่างไร?**
A: ต้องแก้ไขประเทศต้นทางในรายการสินค้าให้ตรงกับที่ระบุในใบอนุญาตค่ะ

**Q: นำเข้าไม่ตรงกับใบอนุญาต ขึ้น error ทำอย่างไร?**
A: ตรวจสอบว่าข้อมูลในใบขนตรงกับใบอนุญาตทุกช่องหรือไม่ ทั้งพิกัด รหัสสถิติ และรายละเอียดสินค้าค่ะ

## 11. ท่าเรือ / เลขยานพาหนะ

**Q: ทำใบกำกับตู้แล้วไม่มีท่าเรือในระบบ เพิ่มเองได้ไหม?**
A: เพิ่มเองไม่ได้ค่ะ ต้องสอบถาม Agent ว่าใช้ท่าเรือรหัสอะไรในระบบ

**Q: VOY เลขมี 019 ทำใบกำกับต้องตัดศูนย์นำหน้าออกไหม?**
A: ไม่ใส่ 0 ข้างหน้าเฉพาะท่าเรือคลองเตย (ท่ากรุงเทพ) เท่านั้น ท่าอื่นใส่ตัวเลขตามปกติ เช่น 019 ค่ะ

**Q: ช่องท่าเรือในใบกำกับไม่ขึ้น ทำอย่างไร?**
A: CDS จะประสานงานกับ NSW เพื่อแก้ไขให้ รอสักครู่ค่ะ ถ้ารอนานให้แจ้งเลขอ้างอิง

**Q: เลขข้างตู้ 45B0 ไม่มีในระบบ ทำอย่างไร?**
A: 45B0 ไม่มีในระบบค่ะ ให้สอบถาม Agent ว่าเค้าส่งเข้าระบบกรมศุลฯเป็นรหัสอะไร

## 12. การเปลี่ยนชื่อเรือ / เที่ยวเรือ

**Q: งานขาออกหากมีเปลี่ยนชื่อเรือ ต้องทำอย่างไร?**
A: ในโปรแกรมไปที่หน้าใบกำกับการขนย้าย → คลิกขวา → พิมพ์ → พิมพ์คำร้องเปลี่ยนชื่อเรือ/เที่ยวเรือ (อันที่ 3) แล้วนำไปให้ชิปปิ้งเดินพิธีการหน้างาน พร้อม B/L ที่ประทับตราให้เอเยนต์รับรองด้วยค่ะ

**Q: ใบขนขาออกชื่อเรือและวอยเรือเปลี่ยน ต้องยกเลิกใบขนแล้วทำใหม่ไหม?**
A: ถ้าเปลี่ยนทั้งชื่อเรือและวอยเรือ ต้องยกเลิกแล้วส่งใหม่ค่ะ ใบขนใหม่จะเป็น 04

## 13. ปัญหาโปรแกรม / การใช้งานทั่วไป

**Q: ประมวลผลนานมากหรือโปรแกรมค้าง ทำอย่างไร?**
A: ติดต่อ 022544027-9 ค่ะ

**Q: ปิดโปรแกรมแล้วเปิดใหม่จะช่วยแก้ปัญหาอะไรได้บ้าง?**
A: ปัญหาหน้าจอค้าง มีสัญลักษณ์แปลก ๆ หรืออยู่ดีๆ ขึ้น error ให้ลองปิดโปรแกรมแล้วเปิดใหม่ก่อนค่ะ มักแก้ปัญหาได้

**Q: สำรองข้อมูลต้องทำทุกวันไหม?**
A: ไม่ต้องทุกวันค่ะ เมื่อโปรแกรมขึ้นแจ้งเตือน "ไม่ได้สำรองข้อมูล" ค่อยกดสำรอง

**Q: พื้นที่สำรองข้อมูลไม่พอ สำรองไม่ได้ ทำอย่างไร?**
A: ตรวจสอบพื้นที่ในไดรฟ์ที่จะสำรองว่าเหลือพอหรือไม่ค่ะ

**Q: โปรแกรมไม่สามารถปรับขนาดช่องได้ไหม?**
A: โปรแกรมเป็นโปรแกรมสำเร็จรูปทำตามโครงสร้างกรมศุล ไม่สามารถเพิ่มขนาดช่องได้ค่ะ

**Q: รายการในใบกำกับหายไปหลังจากแก้ไข ทำอย่างไร?**
A: ให้ลอง Refresh หน้าจอก่อนค่ะ ส่วนใหญ่รายการจะกลับมา สาเหตุมักเกิดจากการใส่สระลอยหรืออักขระพิเศษในข้อมูล

## 14. ลงลายเซ็นต์

**Q: ลงลายเซ็นต์แล้วขึ้น error ทำอย่างไร?**

## 17. ชำระเงิน / ตัดบัญชี

**Q: ลูกค้าตัดบัญชีธนาคารแล้วแต่สถานะในโปรแกรมยังไม่เปลี่ยน ทำอย่างไร?**
A: ต้องตามที่ธนาคารค่ะ ถ้าธนาคารยังไม่ส่งข้อมูลมาที่กรมศุลฯ สถานะก็ยังไม่เปลี่ยน ให้ลูกค้าแจ้งทางธนาคารให้ประสานงานกับกรมศุลด้วย

**Q: ชำระภาษีแล้วสถานะใบขนยังไม่เปลี่ยน ทำอย่างไร?**
A: ให้รีเฟรชหน้าจอใหม่ค่ะ ถ้าเปลี่ยนในใบขนแล้วแต่ในโปรแกรมยังไม่ขึ้น ให้รอหรือรีเฟรชใหม่

**Q: สามารถเสียภาษีโดยตัดบัตรภาษีอิเล็กทรอนิกส์ครึ่งหนึ่งและเงินสดอีกครึ่งได้ไหม?**
A: ไม่ได้ค่ะ บัตรภาษีต้องมีมูลค่ามากกว่าหรือเท่ากับภาษีที่ต้องชำระทั้งหมด

**Q: ใบขนตัดบัญชีครั้งแรก ต้องบันทึกข้อมูลอะไรเพิ่มเติมบ้าง?**
A: ต้องบันทึกข้อมูล "ผู้รับชำระ" ด้วยค่ะ โดยใส่ข้อมูลเดียวกับผู้ชำระ หากไม่มีสาขาก็ต้องเพิ่มสาขาก่อน
